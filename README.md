# G28-STANDALONE-USB-3-AXIS-CONTROLLER
G28 STANDALONE USB 3 AXIS CONTROLLER by Bob8
Use with Arduino Leonardo or ProMicro (ATmega32U).
Install Joystick library

Wiring
DB9 -  ProMicro 
1   -  GND
2   -  A2
3   -  A1
4   -  A0
6   -  VCC
9   -  VCC
